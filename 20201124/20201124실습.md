# 2. 도메인

1. 주소창에 도메인(ex: https://google.com)을 입력
2. 로컬(자신의 컴퓨터) 머신에 등록된 DNS에게 해당 도메인 질의(query)
   1. 해당 DNS에 캐싱되어 있으면 바로 응답
   2. 없으면 상위 네임 서버에 물어봄
3. 응답받은 IP 주소로 통신 (TCP/IP)
